# capstone-data-analytics
Final Capstone Project for Data Analytics Course
##The Impacts on Outside Factors to School Proficiency Scores
This project looked at how median housing value, food insecurity levels, and access to technology outside of school impacted proficiency scores at schools within the city of Phoenix. Tools used within this project were PolicyMap, Google Colab, PowerBI, and github. To reproduce the analysis, download all datasets and the python script.  Using the data sets, run the Python script in Google Colab or your prefered Python notebook service. 
The files are organized by type.  The folders are datasets, scripts, visualizations, and documentation.
